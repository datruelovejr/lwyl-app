# LWYL Typography System

## Font Family
**Primary:** Inter (or system fallback: -apple-system, BlinkMacSystemFont, "Segoe UI", Roboto)

## Font Scale

- **Display (48px)**: Page titles, hero text - `font-size: 48px; font-weight: 700; line-height: 1.2;`
- **H1 (32px)**: Main headings - `font-size: 32px; font-weight: 700; line-height: 1.2;`
- **H2 (24px)**: Section headings - `font-size: 24px; font-weight: 600; line-height: 1.3;`
- **H3 (18px)**: Subsection headings - `font-size: 18px; font-weight: 600; line-height: 1.4;`
- **Body (16px)**: Default text - `font-size: 16px; font-weight: 400; line-height: 1.5;`
- **Small (14px)**: Secondary text, labels - `font-size: 14px; font-weight: 400; line-height: 1.5;`
- **Caption (12px)**: Captions, footnotes - `font-size: 12px; font-weight: 400; line-height: 1.4;`

## Font Weights
- **Regular (400)**: Body text, labels
- **Semibold (600)**: Headings, emphasis
- **Bold (700)**: Display text, strong emphasis

## Colors
- **Primary Text**: `#111827` (near black)
- **Secondary Text**: `#6B7280` (medium gray)
- **Disabled Text**: `#9CA3AF` (light gray)

## Usage
- Use H1 for page titles
- Use H2 for major sections
- Use H3 for subsections
- Use Body for all content
- Use Small for labels and secondary info
- Use Caption for metadata
