# LWYL Brand Colors

## DISC Chart Colors
- **D (Dominance)**: `#C62828` (red)
- **I (Influence)**: `#FFC107` (yellow)
- **S (Steadiness)**: `#4CAF50` (green)
- **C (Compliance)**: `#29B6F6` (blue)
- **Adaptive/Gray bars**: `#9E9E9E` (gray)

## Values/Passion Chart Colors
- **Aesthetic**: `#7CB342` (dark green)
- **Economic**: `#5C8DC4` (dark blue)
- **Individualistic**: `#F28C4E` (dark orange)
- **Political**: `#E05252` (dark red/coral)
- **Altruistic**: `#FFB74D` (dark yellow/gold)
- **Regulatory**: `#757575` (dark gray)
- **Theoretical**: `#B8864A` (dark tan/brown)

## Process/Attributes Chart Colors
- **External**: `#4f92cf` (blue)
- **Internal**: `#C62828` (red)

## UI Colors
- **Primary Action**: `#29B6F6` (DISC blue)
- **Page Background**: `#F9FAFB` (light gray)
- **Card Background**: `#FFFFFF` (white)
- **Primary Text**: `#111827` (near black)
- **Secondary Text**: `#6B7280` (medium gray)

## Usage Rules
- DISC colors ONLY in DISC charts
- Values colors ONLY in Values charts
- Attributes colors ONLY in Attributes charts
- UI colors for buttons, backgrounds, navigation
- Never mix color systems
